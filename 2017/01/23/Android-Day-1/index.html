“#1”

  • Drawable은 현재 컴퓨터에 메모리가 없을 때 사용한다. 현재 내 프로젝트에 이미지가 있으면 쓸 필요가 없다
  • -> SetImageResource
  • FrameLayout에는 하나의 뷰만 사용한다.

“#2”

  • text의 모든 글자는 values.strings.xml에 변수로써 저장 되어야 한다.
  • weightSum은 내부 View에 쓰면 안된다.
  • weight를 사용하면 “0dp”로 Height 나 width를 지정하지 않는다.

“#3”

  • wrap_parent나 “~dp”는 배제한다
  • Table은 자동으로 1:1:.. 비율로 잡힌다
  • |연산자로 두가지 기능을 한번에 담을 수 있다. (양옆에 공백을 만들지 않는다.)

+ArrayAdapter

  • context : App 환경에 대한 총체적 정보를 포함 -> getActivity()
  • ID of list item layout : 우리가 데이터를 표현할 Layout -> R.layout.list_item_forecast
  • ID of textView : 데이터를 표현할 곳의 이름 -> R.id.list_item_forecast_textview
  • list of data : 보여줄 데이터를 의미 -> weekForecast
